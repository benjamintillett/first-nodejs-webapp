How to use
----------

Clone the repository:

```shell 
$ git clone git@github.com:benjamintillett/wherewasi.git
```

And play around with the code

create the database:

```shell 
$ psql
psql=# create database "nodejs-development";
```

Run the server:

```shell 
$ npm start
```
